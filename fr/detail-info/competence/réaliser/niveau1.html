Ressources / Savoirs de référence

R1.01 : Initiation au développement
Technologies : C# , WPF
Contenu : Introduction aux concepts fondamentaux de la programmation (variables, conditions, boucles), structures de contrôle, fonctions, et bases de la programmation. Apprentissage des langages de programmation de base et des outils de développement (Visual Studio).
R1.02 : Développement d'interfaces web
Technologies : HTML , CSS
Contenu : Création d'interfaces utilisateur web avec HTML et CSS. Apprentissage des principes de conception d'interfaces simples, accessibilité de base, et intégration des interactions utilisateur.
R2.01 : Développement orienté objets
Technologies : C#
Contenu : Introduction à la programmation orientée objet : classes (structures de données), objets (instances de classes), héritage (réutilisation de code), encapsulation (protection des données), associations et listes. Structuration de code avec les concepts OOP, modélisation orientée objet de base.
R2.02 : Développement d'applications avec IHM
Technologies : C#
Contenu : Développement d'interfaces utilisateur graphiques (IHM = Interface Homme-Machine), gestion des événements (clics, mouvements), création de composants UI simples, principes d'ergonomie de base.
R2.03 : Qualité de développement
Technologies : Tests Unitaires
Contenu : Bonnes pratiques de développement, tests unitaires simples, qualité du code, maintenabilité.
Éléments observés Analyse
Difficultés rencontrées Premières années d'apprentissage du développement applicatif, compréhension progressive des concepts de base et des outils de développement.
Points de facilité Enthousiasme pour la programmation, curiosité pour les technologies, capacité à assimiler les concepts fondamentaux.
