Ressources / Savoirs de référence

R1.05 : Introduction aux bases de données et SQL
Contenu : Introduction au modèle relationnel (organisation des données en tables), SQL de base (SELECT, INSERT, UPDATE, DELETE), conception de bases de données simples, requêtes avec jointures (combinaison de données de plusieurs tables), MCD, MLD.
R1.06 : Mathématiques discrètes
Contenu : Fondements mathématiques pour la modélisation de données : logique, ensembles, relations, fonctions.
R2.06 : Exploitation d'une base de données
Technologies : SQL, PowerBI
Contenu : SQL avancée, création de vues, exploitation de données avec PowerBI, requêtes complexes.
R2.08 : Outils numériques pour les statistiques descriptives
Technologies : Excel
Contenu : Statistiques descriptives avec Excel, analyse de données.
R2.03 : Qualité de développement
Technologies : Tests Unitaires
Contenu : Bonnes pratiques de développement, tests unitaires simples, qualité du code, maintenabilité.
Éléments observés Analyse
Difficultés rencontrées Évaluation très complexes, beaucoup de notions à assimiler en peu de temps.
Points de facilité Bonne compréhension des concepts de bases de données relationnelles et du langage SQL.
